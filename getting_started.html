Tutorial

This project is maintained by markkimsal

First Request

Start with a git clone into a new directory that is available to your Web server. 

git clone https://github.com/markkimsal/nofw.git mynewproj
Copy the etc/bootstrap.php.txt to etc/bootstrap.php 
cd mynewproj
cp etc/bootstrap.php.txt etc/bootstrap.php
Now let's add a simple hello world file so you can see how the configuration works. Edit the etc/bootstrap.php file so it looks like this: 
<?php
associate_iCanHandle('master', 'nofw/master.php');
associate_iCanHandle('output',  'example/helloworld.php');
Create a directory in src called "example" and create the helloworld.php file 
mkdir src/example

<?php

class Example_Helloworld {

	public function output(&$request, &$response) {
		echo "Hello World. <br/>\n";
	}
}

Not too impressive, is it? Notice how the classname directly maps to the folder/file structure on the disk. Nofw is very opinionated about this structure as it saves a ton of execution time searching the disk and doesn't require auto-loading functionality. With PHP composer you can include other libraries that don't follow this rule, as composer will generate an auto-loading file for you to include for all your dependencies.
